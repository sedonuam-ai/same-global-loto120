# SAME-GLOBAL - LOTO 120

Application PWA de loterie : 9 boutons × 120 grilles fixes.

## Fonctionnalités
- 9 boutons avec 120 grilles chacun (données exactes du fichier Excel)
- Saisie du tirage officiel et vérification des gains
- Barème : 5 bons = 10 paires | 4 = 6 | 3 = 3 | 2 = 1
- Compteurs cumulés par bouton
- **Fonctionne 100% hors-ligne**
- Installable sur Android et iPhone

## Installation GitHub Pages
1. Créer un repo GitHub
2. Uploader tous ces fichiers
3. Settings → Pages → Branch: main → Save
4. Ouvrir l'URL sur mobile → Installer

## Fichiers
- `index.html` — Application principale
- `manifest.json` — Configuration PWA
- `sw.js` — Service Worker (cache hors-ligne)
- `icon-192.svg` / `icon-512.svg` — Icônes

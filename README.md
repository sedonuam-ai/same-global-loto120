# SAME-GLOBAL - LOTO 120

Application PWA : 9 boutons × 120 grilles · Vérification des gains · Compteurs cumulés

## Installation GitHub Pages
1. Uploader tous les fichiers dans un repo GitHub public
2. Settings → Pages → Branch main → Save
3. Ouvrir l'URL sur mobile Chrome → Installer

## Fichiers
- index.html — Application complète
- manifest.json — Config PWA
- sw.js — Service Worker (hors-ligne)
- icon-192.svg / icon-512.svg — Icônes
